    # 通过ip确定位置

    https://api.bilibili.com/x/web-interface/zone

    请求方式：GET

    json回复：

    根对象：

    字段 类型 内容 备注
    code num 返回值 0：成功
    message str 错误信息 默认为0
    ttl num 1
    data obj 信息本体

    data对象：

    字段 类型 内容 备注
    addr str 公网IP地址
    country str 国家/地区名
    province str 省/州 非必须存在项
    city str 城市 非必须存在项
    isp str 运营商名
    latitude num 纬度
    longitude num 经度
    zone_id num ip数据库id
    country_code num 国家/地区代码

    示例：

    curl 'https://api.bilibili.com/x/web-interface/zone'
    查看响应示例：
